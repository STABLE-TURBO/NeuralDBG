# Aquarium IDE Keyboard Shortcuts Reference

## Quick Reference Card

### Essential Shortcuts

| Action | Windows/Linux | macOS | Description |
|--------|---------------|-------|-------------|
| **Parse DSL** | `Ctrl+P` | `⌘P` | Validate current model |
| **Compile** | `Ctrl+B` | `⌘B` | Generate backend code |
| **Run Model** | `Ctrl+R` | `⌘R` | Execute training |
| **Stop Execution** | `Ctrl+C` | `⌘C` | Terminate process |
| **Save Model** | `Ctrl+S` | `⌘S` | Save to file |
| **New Model** | `Ctrl+N` | `⌘N` | Clear editor |
| **Open Model** | `Ctrl+O` | `⌘O` | Load from file |
| **Load Example** | `Ctrl+E` | `⌘E` | Insert example |

## Customization

### Custom Shortcuts

Create custom shortcuts by editing `~/.neural/aquarium/keybindings.json`:

```json
{
  "editor.action.parse": "Ctrl+P",
  "editor.action.compile": "Ctrl+B",
  "editor.action.run": "Ctrl+R",
  "editor.action.stop": "Ctrl+C",
  "editor.action.save": "Ctrl+S",
  "editor.action.loadExample": "Ctrl+E",
  "view.toggleSidebar": "Ctrl+B",
  "view.toggleConsole": "Ctrl+`",
  "navigation.nextTab": "Ctrl+Tab",
  "navigation.previousTab": "Ctrl+Shift+Tab"
}
```

### Conflict Resolution

**If shortcuts don't work**:
1. Check browser doesn't override shortcut
2. Disable browser extensions
3. Check OS keyboard settings
4. Try alternative shortcut
5. Customize in settings

### Platform-Specific Notes

**Windows**:
- `Ctrl` key for most shortcuts
- `Alt` for menu access
- `Win` key may conflict

**macOS**:
- `⌘` (Command) key for most shortcuts
- `⌥` (Option) for alternatives
- `^` (Control) for secondary actions

**Linux**:
- Similar to Windows
- May vary by desktop environment
- Check window manager keybindings

## Quick Tips

### Productivity Shortcuts

1. **Fast Workflow**: `Ctrl+E` → `Ctrl+P` → `Ctrl+B` → `Ctrl+R`
2. **Quick Test**: `Ctrl+E` → `Alt+E` (type 5) → `Ctrl+R`
3. **Export & Open**: `Ctrl+E` → `Enter` → `Ctrl+Shift+O`
4. **Rapid Iteration**: `Ctrl+Z` → Edit → `Ctrl+P` → `Ctrl+R`

### Memory Aids

- **P** = Parse
- **B** = Build (Compile)
- **R** = Run
- **C** = Cancel/Clear
- **E** = Example/Export
- **S** = Save

### Learning Path

1. **Week 1**: Master basic editor shortcuts (Copy, Paste, Undo)
2. **Week 2**: Learn compilation shortcuts (Parse, Compile, Run)
3. **Week 3**: Practice navigation shortcuts (Tabs, Panels)
4. **Week 4**: Explore advanced shortcuts (Debug, Export)
